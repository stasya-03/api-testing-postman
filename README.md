# API Testing (Postman)

Репозиторий с примерами тестирования API.

---

## 📌 Что внутри

- API test cases  
- Примеры запросов (GET, POST, PUT, DELETE)  
- Проверка статус-кодов  
- Проверка response body  
- Позитивные и негативные сценарии  

---

## 🧪 Что тестировалось

- получение данных (GET)  
- создание сущностей (POST)  
- обновление данных (PUT)  
- удаление (DELETE)  

---

## 🛠 Инструменты

- Postman  
- JSON  

---

## 🎯 Цель

Показать навыки API тестирования и анализа ответов сервера.
