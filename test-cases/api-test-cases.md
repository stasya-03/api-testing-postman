# API Test Cases

---

### TC-001 GET request (получение списка пользователей)

**Шаги:**
1. Отправить GET запрос на /users  

**Ожидаемый результат:**
- статус 200  
- возвращается список пользователей  

---

### TC-002 POST request (создание пользователя)

**Шаги:**
1. Отправить POST запрос с валидными данными  

**Ожидаемый результат:**
- статус 201  
- пользователь создан  

---

### TC-003 POST с некорректными данными

**Шаги:**
1. Отправить POST с пустыми полями  

**Ожидаемый результат:**
- статус 400  
- сообщение об ошибке  

---

### TC-004 DELETE request

**Шаги:**
1. Отправить DELETE запрос  

**Ожидаемый результат:**
- статус 200 или 204  
- объект удален
